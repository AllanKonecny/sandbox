This quick example shows that multiple widgets can interfere with one another if care is not taken to avoid attaching multiple widgets to an element. The Sortable widget provides interaction behaviors on the children of a target element. If Draggable is attached to those children after Sortable is instantiated, all the functionality of Sortable is lost. This is probably not an intentional thing to do when constructing a site. However, an incorrect selector can easily grab too many elements and add Draggable which might cause some debugging headaches try to figure out why Sortable is not working as expected.
