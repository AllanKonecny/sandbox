This demo illustrates the orientation option I added to Spinner to provide four layout configurations for the spin controls. The left orientation is shown on the planning wiki and enables adding other controls to the right side of the widget like a drop menu. The vertical and horizontal options add additional flexibility based on how you might want to use the widget. I created three of each in different font sizes to ensure the widget scales properly with the input field.
